Getting started guide

To setup deployment, follow these simple steps:

  1. Create config file touch dogpro.yml
  2. Add configuration from config helper or examples
  3. Commit `dogpro.yml` file
  4. Add project to DOGPRO
  5. Add inventory, you will be presented with public key, add it to deployment server ~/.ssh/authorized_keys.
  6. Open "Commits" tab, select latest commit and press blue arrow to deploy, you should see modal window with deployment configuration
